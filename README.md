# Vehicles in Australia – Warranty & Service Knowledge Base (Structured DITA Project)

## Project Overview

This portfolio project reconstructs a structured technical documentation system for BYD Australia’s Warranty & Service materials, based on publicly available market signals and typical automotive pre-sales documentation needs in Australia.

The goal of this project is to demonstrate:

- DITA-based structured authoring (topic-based architecture)
- Localization workflow using CAT tools (Trados / Phrase)
- Terminology governance and translation asset management
- Version control and documentation lifecycle management
- Pre-sales technical enablement content strategy

This is a professional documentation engineering simulation, not an official BYD publication.

---

## Business Context

In the Australian EV market, warranty and service documentation must:

- Support customer-facing clarity (retail & fleet buyers)
- Align with regulatory and compliance expectations
- Enable consistent messaging across sales, service, and support teams
- Be maintainable across multiple vehicle models and model-year updates

Traditional PDF handbooks are difficult to reuse, version, and localize efficiently.  
This project demonstrates how such materials can be transformed into a modular, structured knowledge base.
